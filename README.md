# PulseFIT

PulseFIT is a modern, responsive landing page for a smartwatch product. 
It is designed with a focus on clean user interface, smooth user experience,and interactive features such as dark mode and mobile navigation.

## Features

- Dark mode toggle with saved user preference
- Fully responsive design across all devices
- Mobile navigation menu (hamburger menu)
- Clean and scalable UI using CSS variables
- Smooth transitions and animations
- Organized and maintainable code structure

## Technologies Used

- HTML5
- CSS3 (Flexbox and Grid)
- JavaScript (Vanilla JS)

## Dark Mode

The application supports both light and dark themes. Users can switch between modes using a toggle button in the navigation bar. The selected theme is stored in the browser using localStorage.
